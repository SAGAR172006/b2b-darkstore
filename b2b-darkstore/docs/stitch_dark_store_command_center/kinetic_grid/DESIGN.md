# Design System Strategy: Cyan Tech

## Visual Narrative
This design system shifts from a dark, neon aesthetic to a luminous, high-energy light mode environment. It maintains a sharp, technical edge through angular geometry while utilizing a vibrant cyan-centric palette that communicates precision, innovation, and clarity. The interface is designed to feel like a high-end digital dashboard under bright clinical light, moving away from the "night-ops" feel of the previous iteration.

## Color Palette
The color system is built on a vibrant cyan foundation, optimized for a light-mode experience.
- **Primary Accent (#00cbfc):** A bright, saturated cyan used for primary actions and brand presence.
- **Secondary Support (#07b7f7):** A slightly deeper azure used for supporting UI components and interactivity.
- **Tertiary Highlight (#5570d5):** A blue-violet accent for badges, notifications, and decorative depth.
- **Neutral Base (#527c95):** A cool, slate-toned blue used for backgrounds and surfaces to maintain the technical atmosphere.

## Typography
The typography pairing balances geometric technicality with functional legibility.
- **Headlines & Labels (Space Grotesk):** A geometric sans-serif that provides a tech-forward, idiosyncratic personality to headings and UI metadata.
- **Body Copy (Inter):** A highly legible, modern sans-serif designed for screen clarity across all sizes.

## Geometry and Spatial Rhythm
- **Sharp Precision (Roundedness 0):** All UI elements utilize hard, 90-degree corners. This brutalist approach emphasizes a rigorous, systematic, and professional tone.
- **Open Layout (Spacing 2):** The density has been adjusted to a normal level, providing sufficient breathing room between elements to allow the vibrant colors to stand out without overwhelming the user. This balance ensures a clean, editorial flow while maintaining a data-rich environment.
